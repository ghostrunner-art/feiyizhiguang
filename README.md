# 项目目标
开发一个名为“非遗之光”的网站，用于全面宣传中国非物质文化遗产。网站需集成AI问答功能，让用户能便捷地了解非遗知识。
# 核心功能
非遗内容展示：
按10个官方类别（民间文学、音乐、舞蹈、戏剧、曲艺、体育游艺杂技、美术、技艺、医药、民俗）分类展示。
提供关键词搜索功能。
每个非遗项目均有包含图文和视频的详情页。
AI智能问答：
网站集成一个AI助手，接入中国非物质文化遗产知识库（开发阶段用模拟数据）。
用户可通过自然语言提问，获取非遗知识。
# 技术栈
语言与框架：Python, Flask
AI模型接口：华为大模型API (代码中需预留接口并提供模拟函数)
数据库：SQLite
